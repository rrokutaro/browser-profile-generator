## ðŸš€ How It Works

1.  **The Script** launches a hidden Linux desktop (Xvfb) and a Chrome browser inside your Codespace.
2.  **NoVNC** creates a web link so you can "see" that desktop from your phone or local computer.
3.  **You** log into YouTube manually (bypassing bot detection).
4.  **The Script** zips up the folder (`~/.config/google-chrome/...`) excluding lock files.
5.  **You** download the zip and use it in your bot.

## ðŸ› ï¸ Step-by-Step Setup

### 1. Run the Generator Script
Click the green code button > codespaces, then create a new codespace and open in vscode web / in browser.

Open your terminal in GitHub Codespaces and run:

```bash
# 1. Make executable
chmod +x get-chrome-profile.sh

# 2. Run with sudo (required for installing dependencies)
sudo ./get-chrome-profile.sh
```

### 2. Connect to the Virtual Desktop
The script will install Chrome/VNC and then pause, printing a message like:
> *Open this URL in your local browser...*

1.  Go to the **PORTS** tab in VS Code.
2.  Look for port **6080**.
3.  Click the **Globe icon** (Open in Browser).
4.  Click **Connect** (no password required).

### 3. Log In Manually
You will see a Linux desktop with Chrome open to YouTube.
1.  Click **Sign In**.
2.  Enter your Google credentials.
3.  **Verify:** Click your profile icon to ensure you are fully logged in and can see your channel dashboard.
4.  **Close the tab** inside the virtual desktop (optional, but good practice).

### 4. Save and Download
1.  Go back to your VS Code terminal.
2.  Press **ENTER** to confirm you are done.
3.  The script will create `chrome-yt-profile.zip`.

#### To Download the Zip (Best Method):
Run this command to start a temporary download server:
```bash
python3 -m http.server 8080
```
1.  Go to the **PORTS** tab again.
2.  Open Port **8080** in your browser.
3.  Click `chrome-yt-profile.zip` to download it to your computer.

## ðŸ¤– How to Use in Puppeteer

Once you have the zip file, unzip it into your bot's project folder so you have a folder named `chrome-yt-profile`.

Here is the code to load it:

```javascript
const puppeteer = require('puppeteer-extra');
const StealthPlugin = require('puppeteer-extra-plugin-stealth');
const path = require('path');

puppeteer.use(StealthPlugin());

(async () => {
  // Path to the unzipped profile folder
  const profilePath = path.resolve(__dirname, 'chrome-yt-profile');

  const browser = await puppeteer.launch({
    headless: false, // Set to true for production
    args: [
      `--user-data-dir=${profilePath}`, // <--- THIS IS THE MAGIC LINE
      '--no-sandbox',
      '--disable-setuid-sandbox'
    ]
  });

  const page = await browser.newPage();
  
  // Go to YouTube studio or homepage
  await page.goto('https://studio.youtube.com');

  console.log('Running... if successful, you should not see a login screen.');
  
  // Do your automation here...

})();
```
